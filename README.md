# replication_code

Replication materials from the [Computational Ethnography Lab](https://github.com/Computational-Ethnography-Lab) for published figures and analyses. Each entry below lists the citation, paper, supplement, code repository, and citation metadata file.

---

**Citation:** Li, Zhuofan, Daniel Dohan, and Corey M. Abramson. 2026. "Temporal Misalignment and Unequal Agency: What Terminal Cancer Patients Teach Us about Time and Inequality." *American Sociological Review*.

**Paper:** https://doi.org/10.1177/00031224261448220

**Supplement:** https://journals.sagepub.com/doi/suppl/10.1177/00031224261448220 (publisher online supplement; Tables A1–A3). The replication repository includes `mca_script.R` and de-identified aggregate data (`mca_data.csv`, 73 participants).

**Code:** https://github.com/lizhuofan95/Replication-Temporal-Misalignment-and-Unequal-Agency  
Clone: `gh repo clone lizhuofan95/Replication-Temporal-Misalignment-and-Unequal-Agency`

**Reference:** [li_2026_asr_temporal_misalignment/CITATION.cff](li_2026_asr_temporal_misalignment/CITATION.cff)

**Citation:** Abramson, Corey M., Tara Prendergast, Zhuofan Li, and Daniel Dohan. 2026. "Qualitative Research in an Era of Artificial Intelligence: A Pragmatic Approach to Data Analysis, Workflow, and Computation." *Annual Review of Sociology* 52.

**Paper:** https://doi.org/10.1146/annurev-soc-011824-104836

**Supplement:** https://www.annualreviews.org/content/journals/10.1146/annurev-soc-011824-104836#supplementary_data

**Code:** https://github.com/Computational-Ethnography-Lab/cmap_visualization_toolkit  
Clone: `gh repo clone Computational-Ethnography-Lab/cmap_visualization_toolkit`

**Reference:** [abramson_2026_ars_cmap_visualization/CITATION.cff](abramson_2026_ars_cmap_visualization/CITATION.cff)

**Citation:** Arteaga, Ignacia, Alma Hernandez de Jesus, Brandi Ginn, Corey M. Abramson, and Daniel Dohan. 2025. "Understanding how social context shapes decisions to seek institutional care: a qualitative study of experiences of progressive cognitive decline among Latinx families." *The Gerontologist* 65(11):gnaf207.

**Paper:** https://doi.org/10.1093/geront/gnaf207

**Supplement:** https://academic.oup.com/gerontologist/article/65/11/gnaf207/8256431#supplementary-data (includes R replication code and clustering details). Open-access article: https://pmc.ncbi.nlm.nih.gov/articles/PMC12578513/

**Code:** [arteaga_2025_latinx_heatmap/](arteaga_2025_latinx_heatmap/) in this repository (`latinx_array_final.R` at repository root and in that folder).  
Clone: `gh repo clone Computational-Ethnography-Lab/replication_code`

**Reference:** [arteaga_2025_latinx_heatmap/CITATION.cff](arteaga_2025_latinx_heatmap/CITATION.cff)

**Reproduce (after cloning this repository):**

```bash
cd arteaga_2025_latinx_heatmap
Rscript latinx_array_final.R
```
